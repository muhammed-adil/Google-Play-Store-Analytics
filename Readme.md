# Google Play Store Data Analysis

## Project Overview
This project analyzes Google Play Store data using Python and Power BI.
Multiple visualizations were created to analyze installs, ratings,
categories, revenue, and trends.

## Tools Used
- Python
- Pandas
- Plotly
- Power BI

## Tasks
1. Bubble Chart
2. Choropleth Map
3. Time Series Chart
4. Stacked Area Chart
5. Grouped Bar Chart
6. Dual Axis Chart

## Dataset
Google Play Store dataset provided during training.
